# 🤖 AI Facebook Automation with n8n

An AI-powered automation workflow built with **n8n**, **Google Gemini**, and the **Meta Graph API** to automatically manage Facebook Messenger conversations and Facebook comments.

## ✨ Features

- 💬 Automatic replies to Facebook Messenger messages.
- 💭 AI-powered Facebook comment analysis.
- 🤖 Intelligent responses using Google Gemini.
- 🖼️ Image understanding.
- 🎙️ Voice message transcription.
- 📚 Custom Knowledge Base integration.
- ⚡ Webhook-based real-time automation.
- 🔀 Smart routing based on message type.

## 🛠️ Tech Stack

- n8n
- Google Gemini
- Meta Graph API
- Webhooks
- HTTP Request
- AI Agent
- JSON
- Google Sheets (Knowledge Base)

## 📂 Workflow

The workflow includes:

- Webhook Trigger
- Message Type Classification
- Image Download & Analysis
- Voice Download & Transcription
- AI Agent
- Response Parsing
- Messenger Auto Reply
- Facebook Comment Reply

## 🚀 Getting Started

1. Clone this repository.
2. Import the workflow JSON into n8n.
3. Configure your credentials:
   - Meta Graph API
   - Google Gemini API
4. Update your Webhook URL.
5. Activate the workflow.
